# 🍎 果园病虫害风险预警与数据要素可视化平台

> **数据要素赛道作品** | LightGBM + SHAP 智能预警 | Streamlit 企业级大屏

## 一、项目定位

本项目面向果园病虫害"防早、防准、防省"的业务痛点，构建了一条完整的**数据要素价值链**：多源数据集成 → 标准化特征工程 → ML 风险预警模型 → POSI 精准防控方案 → 数据服务输出 → 降本增收价值闭环。

核心技术创新：
- **LightGBM 三分类模型**：13 维环境/生物特征 → 低/中/高三级风险概率预测（宏平均 F1=0.978, AUC=0.9997）
- **SHAP 可解释性**：每个地块的风险判据可追溯至具体环境因子贡献，搭配农业通俗诊断文案 + 靶向调控建议
- **POSI 精准防控指数**：7 因子加权 + Youden 阈值标定，判定每地块是否进入"防治窗口"
- **RRI 区域风险分级**：Jenks 自然断点法将 16 个防控单元划分为红/黄/绿三级响应区
- **数据资产价值链全景**：从原始多源数据台账到降本增收效益量化的完整运营视图

## 二、项目架构

```
多源数据 (气象/土壤/虫情/管护)
  └─→ data_process/ 加工脚本（特征工程 + LightGBM 建模 + POSI 防控）
       ├─→ uploads/data.csv       原始 300 地块记录
       ├─→ 1.预处理/              数据挖掘 + 标准化
       ├─→ 2.风险划分/            LightGBM 三分类 + SHAP
       └─→ 分区域分时段/          RRI + POSI + 帕累托
            └─→ output/           最终模型产出（24 个 CSV + 22 张图表）
                 └─→ app.py       Streamlit 可视化大屏
```

## 四、环境部署

### 4.1 环境要求
- Python 3.10 ~ 3.12
- Windows / macOS / Linux

### 4.2 快速启动

```bash
# 1. 克隆项目
cd crop

# 2. 安装依赖（推荐使用虚拟环境）
pip install -r requirements.txt

# 3. 启动看板
streamlit run app.py

# 4. 浏览器自动打开 http://localhost:8501
```

### 4.3 演示模式（无原始数据时使用）

项目内置 `demo_data/` 脱敏示范数据集，评委可离线直接运行。代码自动检测 `output/` 目录是否存在，若缺失则回退到 `demo_data/`。

### 6.2 数据性质说明

| 数据类别 | 来源 | 性质 |
|---|---|---|
| 原始多源数据 | `uploads/data.csv` | 300 地块脱敏合成数据，模拟真实果园气象/土壤/虫情/管护参数 |
| 加工脚本 | `data_process/代码/` | 完整 Python 处理链路（5 个脚本，1953 行），可独立复现 |
| 特征矩阵 | `output/1.预处理及特征工程结果/` | 脚本输出，包含衍生指数 + 标准化矩阵 |
| 模型训练产出（KPI/SHAP/混淆矩阵） | `output/2.低中高/tables/` | LightGBM 真实建模结果，非仿真 |
| 业务决策产出（RRI/POSI/防控方案） | `output/3.分区域分时段/tables/` | 算法推导结果，基于模型输出计算 |
| 数据资产运营指标（接入量/调用量/效益） | `utils/data_loader.py` `generate_ops_metrics()` | **仿真演示数据**，固定种子可复现，贴合作物季节波动规律，降本增收参数复用 100 亩果园测算口径 |
| 分级防控效益量化参数 | `modules/m05_three_tier_response.py` | **仿真演示数据**，基准农药 78 元/亩·次、亩产值 3,000 元/亩 |

> ⚠️ **仿真边界说明**：数据资产运营中心（Module 8）和防控效益量化（Module 5）中的月度接入量、服务调用次数、降本增收金额等指标为固定随机种子仿真生成，用于展示"数据→价值"的完整链路逻辑，非真实业务统计数据。模型预测、SHAP 分析、防控方案推荐均基于真实算法输出。

## 七、项目创新点

1. **数据要素价值链完整闭环**：从原始多源数据接入 → 特征资产沉淀 → 数据产品产出 → 服务调用 → 价值量化，全链路可视化运营
2. **降本增收可量化**：分级防控效益参数（农药减量 35%~94%、亩均节约 42~73 元），让 AI 预警的经济价值透明可算
3. **可解释 AI + 农业通俗诊断**：SHAP 特征贡献自动翻译为大白话诊断文案 + 靶向田间农事建议，让农技人员而非数据科学家看得懂
4. **三级差异化防控响应**：红/黄/绿三区从应急施药到仅监测的梯度策略，避免一刀切用药
5. **POSI 精准防控指数**：融合 7 个环境因子的加权判定，仅对"窗口内"地块施药，避免盲目全覆盖
